Kernel Control-Flow Protection based on eBPF.

## Building

```sh
$ git clone https://github.com/CrackLewis/kcfp-ebpf
$ cd kcfp-ebpf
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## Running 

```sh
$ sudo ./kcfp_ebpf [SYSCALLS]
```
